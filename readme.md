## ✨ Features

### 📐 Core Mathematics Engine
- Solve equations (algebraic, polynomial, trigonometric)
- Step-by-step solutions
- Verification of answers

### ∫ Calculus
- Definite & indefinite integration
- Full step breakdown
- Graph visualization of area

### ∂ Differentiation
- First to nth-order derivatives
- Critical points detection
- Function plotting

### lim Limits
- One-sided & two-sided limits
- Graph-based visualization

### 📊 Matrix Algebra
- Determinant
- Inverse
- Eigenvalues
- Rank & trace

### 🔐 Number Theory
- Prime checking
- Factorization
- Euler’s Totient function
- Divisors

### 📈 Graphing System
- 2D function plotting
- 3D surface plotting (Plotly)
- Multiple function comparison

### 🔢 Series Expansion
- Taylor / Maclaurin series
- Function approximation visualization

### 🤖 AI Assistant (Groq LLM)
- Step-by-step mathematical explanations
- Intuition-based teaching
- Advanced theory support (Algebra, Calculus, Number Theory)

### 🔭 Wolfram Alpha Integration
- External computational knowledge engine

---

## 🧠 Tech Stack

- Streamlit (UI framework)
- SymPy (Symbolic mathematics engine)
- NumPy (Numerical computing)
- Plotly (Interactive graphs)
- Matplotlib (support plotting)
- Groq API (LLM reasoning)
- Wolfram Alpha API (external math engine)
